DayaMel-Cuidado de la piel

Requisitos previos

Antes de ejecutar el proyecto, asegúrese de tener instalado:

PHP 8.0 o superior.
MySQL o MariaDB.
Servidor web Apache (XAMPP, Laragon o WAMP).
Navegador web actualizado.
Instalación

Descargar o clonar el proyecto.
Copiar la carpeta del proyecto dentro del directorio del servidor web.
Crear una base de datos llamada "dayamel_db".
Importar el archivo "dayamel_db.sql" incluido en el proyecto.
Configurar los datos de conexión en el archivo:
config/conexion.php

Iniciar Apache y MySQL.
Ejecución

Abrir el navegador y acceder a:

http://localhost/Proyecto-DayaMel/public/index.php?action=inicio
