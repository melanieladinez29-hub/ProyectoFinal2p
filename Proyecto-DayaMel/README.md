DayaMel-Skincare

Requisitos previos

Antes de ejecutar el proyecto, asegúrese de tener instalado:

- PHP 8.0 o superior.
- MySQL o MariaDB.
- Servidor web Apache (XAMPP, Laragon o WAMP).
- Navegador web actualizado.

Instalación

1. Descargar o clonar el proyecto.
2. Copiar la carpeta del proyecto dentro del directorio del servidor web.
3. Crear una base de datos llamada "dayamel_db".
4. Importar el archivo "dayamel_db.sql" incluido en el proyecto.
5. Configurar los datos de conexión en el archivo:

config/conexion.php

6. Iniciar Apache y MySQL.

Ejecución

Abrir el navegador y acceder a:

http://localhost/Proyecto-DayaMel/public/

Funcionalidades principales

- Catálogo de productos.
- Carrito de compras.
- Formulario de contacto.
- Gestión de productos.
- Gestión de categorías.
- Gestión de clientes.
- Gestión de proveedores.
- Panel administrativo.
